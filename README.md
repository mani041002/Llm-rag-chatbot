# LLM RAG Chatbot

Chat with your PDFs using Retrieval-Augmented Generation (RAG).

## 🚀 Features
- Upload a PDF and ask questions
- Semantic search with FAISS
- LLM-powered answers (Mistral API)
- Simple Streamlit chat UI

## 🧰 Tech Stack
- Python
- Streamlit
- Mistral API
- Sentence Transformers
- FAISS
